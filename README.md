# Gallery+

Aplicação fullstack de galeria de imagens desenvolvida com React, TypeScript e Node.js, com back-end próprio e upload de fotos.

## Tecnologias

- [React](https://react.dev/) — biblioteca para construção de interfaces
- [TypeScript](https://www.typescriptlang.org/) — tipagem estática
- [Vite](https://vitejs.dev/) — bundler e servidor de desenvolvimento
- [Tailwind CSS](https://tailwindcss.com/) — estilização
- [Node.js](https://nodejs.org/) — servidor back-end
- [tsup](https://tsup.egoist.dev/) — bundler do servidor

## Funcionalidades

- Upload de imagens para a galeria
- Listagem de imagens cadastradas
- Back-end próprio com API REST
- Coleção Postman incluída para teste das rotas

## Como Executar

```bash
pnpm install

# Em terminais separados:
pnpm dev-server   # inicia o back-end
pnpm dev          # inicia o front-end
```

## Sobre

Projeto fullstack desenvolvido para praticar a integração entre front-end em React e back-end em Node.js com TypeScript.
